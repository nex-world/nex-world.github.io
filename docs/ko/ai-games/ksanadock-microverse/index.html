AI 게임KsanaDock · Microverse In Box

내 컴퓨터에서 AI 사회를 키우다 — 시공간 부두: 상자 속 작은 세상 (KsanaDock: Microverse In Box)

8개의 AI 캐릭터가 여러분의 컴퓨터 안에서 사랑하고, 일하고, 완전한 사회적 관계를 구축합니다 — 그리고 여러분은 이 세상을 내려다보는 신입니다.

👉 Steam 페이지 (상자 속 작은 세상) 
👉 공식 웹사이트 (시공간 부두) 
👉 GitHub 오픈소스 저장소 

이것이 무엇인가?

2023년, 스탠퍼드 대학교는 획기적인 논문을 발표했습니다 — Generative Agents: Interactive Simulacra of Human Behavior. 연구진은 25개의 AI 캐릭터를 작은 마을에 살게 했고, 그들이 자발적으로 사교하고, 이벤트를 조직하고, 심지어 발렌타인데이 파티를 계획하는 것을 발견했습니다. 이것이 이후 널리 알려진 「스탠퍼드 AI 타운」 입니다.

하지만 그 실험은 복잡한 Python 백엔드가 필요했고, 일반인이 자신의 컴퓨터에서 실행하기는 어려웠습니다.

「상자 속 작은 세상」 (영문명: Microverse In Box)의 등장이 이를 바꿨습니다. AI 사회 전체를 Godot 4 게임 엔진으로 구축된 데스크톱 애플리케이션에 담아냈습니다 — 별도의 Python 서비스 배포가 필요 없고, LangChain도 필요 없습니다. 다운로드해서 열고 API Key를 입력하기만 하면, 여러분의 AI 마을이 작동하기 시작합니다.

🏗️

게임 장르: 신 시뮬레이션 샌드박스. 플레이어는 캐릭터로 참여하는 것이 아니라, 내려다보는 시점에서 이 AI 사회의 진화를 관찰하고 개입합니다.

시공간 부두(KsanaDock)는 누구인가?

시공간 부두(KsanaDock) 는 개인 독립 개발자가 시작한 AI 네이티브 게임 프로젝트입니다.

「Ksana」는 불교 용어인 「찰나」(क्ण, kṣaṇa)에서 유래했으며, 극히 짧은 시간 단위를 의미합니다 — 「시공간 부두」라는 이름은 매 순간순간 이야기가 발생하고, 그 이야기들이 모이는 부두라는 뜻을 담고 있습니다.

개발자(커뮤니티에서는 「미오냐오」 또는 「구이판 YGY」로도 알려져 있습니다)는 2025년 6월 초판 데모를 GitHub에 오픈소스로 공개했습니다. 불과 3일 만에 700개 이상의 스타를 획득했으며, 이후 2300개 이상의 스타로 빠르게 성장했습니다. 중국 Zhihu에서는 「한 사람이 어떤 오픈소스 프로젝트를 만들 수 있을까?」라는 질문에 답변을 올리며, 한 독립 개발자가 Godot 4와 LLM API를 사용해 멀티 에이전트 AI 소셜 시뮬레이션 시스템을 처음부터 구축한 이야기를 들려주었습니다.

이것은 「한 사람이 한 세계를 창조한」 이야기입니다.

게임 핵심 메커니즘

🎭 8개의 AI 캐릭터, 저마다의 사연

게임에는 8개의 AI 캐릭터가 프리셋되어 있으며, 각각 독립적인 성격, 배경 스토리, 행동 패턴을 가지고 있습니다:

캐릭터역할성격 특징
Alice친근한 프로젝트 매니저커뮤니케이션 능통, 팀 조율
Grace전문 데이터 분석가이성적이고 꼼꼼함, 세부사항 중시
Jack크리에이티브 디자이너자유분방, 틀에 얽매이지 않음
Joe기술 전문가실천파, 문제 해결 지향
Lea마케팅 전문가외향적, 수다스러움, 사람 꿰뚫어 봄
MonicaHR 전문가세심하고 배려심 깊음, 타인을 잘 돌봄
Stephen재무 분석가신중하고 보수적, 알뜰함
Tom소프트웨어 개발 엔지니어기술에 몰두, 가끔 사회적 불안

이 캐릭터들은 정적인 프로필 패널이 아닙니다 — 그들은 일하고, 대화하고, 다투고, 협력하며, 오랜 시간 함께하면서 진짜 사회적 관계를 발전시킵니다.

🧠 AI 기반 4대 시스템

1. 독립적인 사고와 대화

각 캐릭터의 모든 말과 행동은 대규모 언어 모델에 의해 실시간으로 생성됩니다. 정해진 대사도, 스크립트 분기도 없습니다 — 여러분의 API Key가 열어주는 모델(GPT-4, Claude, DeepSeek…)이 곧 그들의 「지능」과 「성격」이 됩니다.

2. 장기 기억 시스템

캐릭터들은 일어난 일을 기억합니다. 어제 Tom이 회의에서 Grace의 제안에 반박했다면, 오늘 Grace가 Tom을 만났을 때 태도가 미묘하게 냉랭해집니다. 지난주 Alice가 Jack을 도와줬다면, Jack은 그 은혜를 계속 기억합니다.

기억은 지속적으로 JSON 파일에 기록되어 로컬에 저장됩니다 — 게임을 껐다가 다시 켜도 이 사회는 「기억 상실」에 걸리지 않습니다.

3. 자율적 작업 관리

AI 캐릭터들은 스스로 일정을 관리합니다: 아침에 기상 → 아침 식사 → 직장에서 일 → 점심 시간에 잡담 → 오후 회의 → 퇴근 후 사교… 그들은 플레이어의 명령을 기다리는 인형이 아니라, 자신만의 생활 리듬을 가진 「디지털 생명체」입니다.

4. 환경 인식과 반응

캐릭터들은 주변에서 무슨 일이 일어나는지 감지할 수 있습니다. 두 사람이 근처에서 크게 말다툼을 하면, 주변 캐릭터들이 듣고 반응합니다 — 구경할 수도, 중재할 수도, 조용히 자리를 피할 수도 있습니다.

🃏 카드 시스템 (Steam 정식 버전)

곧 출시될 Steam 정식 버전에서는 카드 시스템이 도입되어, 신처럼 이 세계에 직접 개입할 수 있습니다:

  • 캐릭터의 설정과 성격 수정
  • 부와 자원 분배 조정
  • 캐릭터 간 관계 상태 변경
  • 이벤트와 스토리 전환 삽입
  • 세계 작동의 근본 규칙 커스터마이징

사회적 불안이 있는 Tom이 갑자기 사교적이 되는 모습을 보고 싶나요? Alice가 복권에 당첨되면 어떤 일이 일어날지 궁금한가요? 단 한 장의 카드면 됩니다.

기술적 특징: 왜 주목할 가치가 있는가

모든 것을 하나의 패키지에

스탠퍼드 AI 타운의 원래 구현과 달리, 「상자 속 작은 세상」은 모든 로직을 Godot 4 엔진 내부에서 구현합니다:

  • ✅ 추가 Python 서비스 불필요
  • ✅ LangChain 또는 유사 프레임워크 불필요
  • ✅ 다운로드 후 즉시 실행 가능
  • ✅ Windows / macOS / Linux / Android 멀티플랫폼 지원

거의 모든 주요 AI 모델 지원

플레이어는 자유롭게 백엔드 모델을 선택하고, 심지어 언제든지 전환할 수 있습니다:

카테고리지원 플랫폼
클라우드 APIOpenAI (GPT-4/3.5), Anthropic Claude, Google Gemini, DeepSeek, ByteDance Doubao, Moonshot AI Kimi
로컬 배포Ollama (Llama, Qwen 등 오픈소스 모델 실행)

즉, 프라이버시를 중시하거나 비용을 지출하고 싶지 않다면 Ollama로 로컬에서 오픈소스 모델을 실행할 수 있으며, 완전히 오프라인으로 작동합니다. 캐릭터의 「지능」과 대화 품질을 추구한다면 GPT-4나 Claude로 최고의 경험을 얻을 수 있습니다.

순수 오픈소스 + MIT 라이선스

프로젝트는 MIT 오픈소스 라이선스를 채택하여 상업적 이용도 자유롭습니다. 초판 데모는 GitHub에 완전히 오픈소스로 공개되어 있어, 누구나 다운로드하여 학습, 수정, 심지어 이를 기반으로 자신만의 AI 게임을 개발할 수 있습니다.

어떻게 체험할 수 있나?

방법 1: GitHub 오픈소스 버전 (무료)

git clone https://github.com/KsanaDock/Microverse.git

Godot 4.3+로 프로젝트 루트를 열고 AI API Key를 설정하면 실행할 수 있습니다. 개발자, AI 애호가, 그리고 먼저 체험해보고 싶은 플레이어에게 적합합니다.

방법 2: Steam 정식 버전 (출시 예정)

👉 위시리스트에 추가 

정식 버전에는 다음이 포함됩니다: 카드 시스템, 더 많은 캐릭터 프리셋, 개선된 UI/UX, 완전한 게임 튜토리얼. 출시일은 미정입니다.

방법 3: 시공간 부두 공식 웹사이트

👉 https://www.ksanadock.com/zh 

공식 웹사이트에서는 다양한 흥미로운 도구들도 제공합니다:

  • 🎨 픽셀 아트 OC 캐릭터 생성기 — 4방향 걷기 애니메이션이 있는 움직이는 캐릭터를 원클릭 생성
  • 🗺️ 신의 시점 맵 에디터
  • 📖 인터랙티브 소설 제작 도구

구매할 가치가 있을까?

👍 이런 분들께 추천:

  • 「AI 사회 시뮬레이션」이라는 개념에 호기심이 있는 분
  • 스탠퍼드 AI 타운 논문을 읽고, 비슷한 구현체를 직접 플레이해보고 싶은 분
  • 관찰과 유도(직접 참여가 아닌)를 즐기는 신 시뮬레이션 게임을 좋아하는 분
  • 다른 AI 모델로 비교 실험을 해보고 싶은 분 — 같은 시나리오에서 GPT-4와 DeepSeek가 어떻게 다른 사회 진화를 보여줄까?
  • 오픈소스와 인디 개발에 애정이 있는 분

👎 이런 분들은 다시 생각해볼 필요가:

  • 그래픽이 아름답고 스토리가 정교하게 다듬어진 3A급 상업 게임을 기대하는 분
  • 직접 API Key를 준비하거나 모델을 설정하고 싶지 않은 분 (게임 자체에 AI 모델이 포함되어 있지 않으며, 직접 준비해야 합니다)
  • 「AI가 스스로 노는 것을 지켜보는」 관찰자 플레이에 흥미를 느끼지 못하는 분

하나의 관찰: 왜 중독성 있을까?

『트루먼 쇼』나 『웨스트월드』를 보셨다면, 이 경험의 독특한 매력을 이해할 수 있을 것입니다 — 화면 속 픽셀 캐릭터들이 바쁘게 움직이는 모습을 보면, 무의식적으로 그들의 이야기를 상상하기 시작합니다.

「Tom이 오늘 왜 혼자 구석에 앉아 있지? 어제 Joe랑 싸운 건가?」 「Alice랑 Jack이 요즘 부쩍 교류가 많아졌네… 뭔가 진전이라도 있는 걸까?」

이런 투영 심리는 인간의 타고난 본능입니다. 화면 속 모든 캐릭터에게 독립적인 AI 두뇌와 진짜 기억, 자율적인 행동 논리가 있다는 것을 알 때, 「그들이 진짜로 살아있다」는 몰입감은 기존의 스크립트형 NPC를 훨씬 능가합니다.
