NexTavern

NexTavern 完全ユーザーガイド

プラットフォーム概要

NexTavern（旧 Tavern Lite）は、AI を活用したロールプレイ＆チャットプラットフォームです。好みのキャラクターを作成し、大規模言語モデル（LLM）による知的で自然な会話を楽しめます。

シンプルなチャットボットとは異なり、目標達成型のゲームモードや複数キャラクターの会話など、多彩な遊び方を提供します。気軽なロールプレイから本格的なファンタジー冒険まで、さまざまなニーズにお応えします。

クイックスタート

ステップ1：AI モデルの設定

  1. 上部ナビゲーションの 「Go」→「Config」 をクリック。
  2. 左メニューの 「設定」→「LLM 設定」 を開く。
  3. API Key を取得し（DeepSeek 推奨）、「API Key」欄に貼り付けて 「Fetch Models」 をクリック。
  4. モデルを選択（例：deepseek-chat）すれば準備完了。

ステップ2：キャラクターを作成

  1. 「新規キャラクター」 をクリック。
  2. 名前、外見、性格、背景を設定。
  3. 最初の 「挨拶文」 を書いて保存。
  4. 「広場」 でキャラクターを選び、「会話を開始」 をクリック。

コア機能

1. キャラクター管理

  • 工房：キャラクターの作成・編集センター。すべての設定を細かく調整できます。
  • 広場：作成済みの全キャラクターを閲覧可能。検索やタグフィルターに対応し、直接会話を開始できます。

2. セッション管理

すべての会話履歴は 「セッション」 ページに保存されます。過去の会話を続行したり、セッション名の変更や削除、会話ログのエクスポートも可能です。

3. 特色あるゲームモード

チャレンジモード — 目標に向けた会話

特定の目標達成を目指すロールプレイ。「国王の信頼を得る」「真犯人を暴く」など、目標を設定して会話を進めます。AI が進行状況を評価し、達成感のあるストーリー体験を提供します。

  • 仕組み：目標に応じて「信頼値」「疑念度」などの隠しステータスが設定されます。キャラクターとの会話ごとに AI が評価し、これらのステータスが変動します。
  • 魅力：会話中に進捗を示すヒント（「王はやや心を動かされたが、まだ決断していない」など）が表示されます。戦略的に会話を選択して目標を達成したときの達成感は格別です。
DnD モード — テーブルトークRPG体験

テーブルトークRPG の完全デジタルシミュレーション。キャラクターの能力値を設定し、AI がダンジョンマスター（DM）となって冒険を導きます。

  • 仕組み：「筋力」「敏捷力」「知力」などの6属性を持つキャラクターシートを作成。AI が DM としてストーリーを進行します。
  • 中核メカニズム — ダイスロール：キャラクターが行動を起こすとき（「忍び寄って衛兵をかわす」「商人を説得する」など）、システムが仮想の20面ダイス（D20）をロールし、キャラクターの属性と組み合わせて成否を判定します。DM が結果に基づいて意外な展開を描写します。
  • 魅力：本格的なターン制探索、謎解き、戦闘を体験。ダイスのランダム性が毎回のアクションに驚きをもたらします。複雑なキャラクター成長と戦略的な連携が、深いリプレイ価値を提供します。
グループチャットモード — 複数キャラクターの交流会

複数の AI キャラクターを同じ会話に集めて、交流を観察または参加できます。

  • 仕組み：キャラクターライブラリから複数のキャラクターを選択してグループチャットを作成。自分の「素」で発言することも、静観することもできます。
  • 魅力：AI が自動で発言順を調整し、自然で活気のある会話が生まれます。異なる性格のキャラクター同士が衝突し、ドラマチックな展開が自発的に生まれる様子は、クリエイティブな刺激に満ちています。

体験を最適化する

  1. AI の応答スタイルを調整：「LLM 設定」の 「Temperature」 パラメーターを調整してみてください。低めの値（0.2 程度）では安定した応答に、高めの値（0.8 程度）ではより創造的で変化に富んだ応答になります。
  2. キャラクターのパフォーマンスを向上：AI の応答が期待通りでない場合は、「工房」でキャラクターを編集し、より詳細な性格描写や具体的な会話例を追加してください。これによりロールプレイの精度が大幅に向上します。
  3. データのバックアップ：「工房」のエクスポート機能で大切なキャラクターカードを、「セッション」のエクスポート機能で会話記録を定期的にバックアップしましょう。

よくある質問とトラブルシューティング

  • AI が応答しない、またはエラーが出る：まず「LLM 設定」の API Key が正しく、有効期限が切れていないか確認してください。ネットワーク接続もご確認ください。
  • キャラクターから外れた応答が返ってくる：キャラクターにより明確な「システムプロンプト」（例：「常に冷たい口調で話してください」）を追加してみてください。また、キャラクターの「性格設定」を見直し、充実させることも効果的です。
  • 応答を再生成したい：会話画面で AI の応答にマウスカーソルを合わせると、「再生成」ボタンが表示されます。クリックすると新しい応答が生成されます。
  • ページの動作が遅い：ブラウザのキャッシュをクリアするか、使用していない会話タブを閉じてみてください。

最後に

一番の上達方法は、すぐに実践することです。まずはシンプルなキャラクターを作成して、基本的なチャットを楽しんでください。その後、チャレンジモードで目標達成の爽快感を味わったり、グループチャットでにぎやかな交流を体験したりしてみましょう。準備ができたら、DnD モードに足を踏み入れ、真の伝説的冒険を始めてください。

NexTavern の世界で、数え切れないほどの素晴らしい物語が生まれますように。
