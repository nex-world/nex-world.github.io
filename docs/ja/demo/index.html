NEXWORLD Demo

NEXWORLD Demo を試す

この Demo では、AI が動的なテキストアドベンチャー世界をどのように動かすのかを体験できます。あなたの選択は単なる分岐ではなく、物語の状態に影響します。

物語の導入

あなたは突然、修仙の世界で目を覚まします。江城の城主の娘・岳炎炎として、家柄と才能に恵まれ、これまで好き放題に振る舞ってきました。しかし今、見知らぬ記憶が怒涛のように押し寄せます——あなたは自分がある小説の脇役であり、主人公に倒される運命の悪役であることを恐怖とともに悟ります。

決闘まで残された時間は、わずか三日。

運命を受け入れるのか、それともこの「予知」の記憶を頼りに結末を書き換えるのか。この世界は、あなたの抗いにどう応えるのでしょうか。

三日間の生存之旅を始める

重要な注意点：Demo の実行には AI モデルとの接続が必要です。初回利用時は、画面下部の 「モデル設定」 セクションに有効な API Key を入力してください。ご不明な点は、以下のプレイヤーコミュニティまでお気軽にお問い合わせください。

あわせて体験：Tavern AI ロールプレイプラットフォーム

Demo で AI 世界との対話に魅力を感じ、より自由度の高いキャラクターやストーリー創作に興味が湧いた方には、もうひとつのツールをご用意しています。

Tavern は軽量で使いやすい AI ロールプレイプラットフォームです。ストーリー重視の Demo とは異なり、Tavern は完全なクリエイター権限をあなたに与えます。

Tavern では、以下のことが可能です：

  • どんなキャラクターでも作成：ファンタジーの英雄から現代のアイドルまで、外見、性格、背景を詳細に設定。
  • 多彩なプレイモード：
    • フリーチャット：キャラクターと自由に会話。
    • チャレンジモード：「信頼を得る」などの目標を設定し、会話で達成を目指す。
    • DnD モード：キャラクター属性とダイスを使った本格的なテーブルトークRPG。
    • グループチャットモード：複数の AI キャラクターを同じ会話に集めて交流を楽しむ。
  • 完全なコントロール：すべての会話履歴とキャラクターデータはお使いのデバイスに保存され、あなたが完全に管理できます。

Tavern は、想像力を解き放ち、AI ロールプレイの多様な楽しみ方を体験するための絶好のサンドボックスです。
