AI 游戏推荐AgenTank：AI 坦克大战

AgenTank：不是你在打游戏，是你的坦克 Agent 在打

你给 AI 一个”战术思路”，AI 把它写成坦克的控制代码，然后你的坦克就上战场了——7×24 小时不眠不休，在你睡觉的时候也在偷偷上分。
450 万场对局、80+ 王者段位坦克、月运营成本不到 100 元——这是目前最离谱的 AI 原生游戏之一。

👉 官网（网页即玩） 

这不是一个”游戏”

在聊 AgenTank 之前，先把话说清楚：这不是一个传统意义上的游戏。

传统游戏里，你按 W 前进、按空格射击、用鼠标瞄准——你的操作直接映射为游戏行为。

AgenTank 不是这样的。

你的操作是——打字。

  • “打不过就绕后，利用草丛隐身，等敌人技能 CD 好了再打。”
  • “前期苟着发育，看到残血再冲出去收割。”
  • “如果对方坦克血量低于 30%，直接传送到他脸上放眩晕。”

你把这些思路写下来，AI（Claude Opus、GLM 等大模型）把你的话翻译成坦克的控制脚本——然后你的坦克就上战场了。

没错，你自己不需要上场。你的敌人也不是人——是别人的 AI 坦克。

这就是为什么玩家管它叫 “赛博斗蛐蛐”。

🎮

一句话定义：你当教练，AI 当选手，浏览器就是赛场。

为什么它会存在？

这个故事要从开发者 DK 说起。

2016 年，他参加了一个代码对战比赛，用 600 行 Ruby 代码写了一个坦克 AI，打进了十六强。那种”自己的代码在战场上厮杀”的感觉，让他念念不忘。

2021 年他想复刻这个体验，但发现写一个完整的游戏 + AI 系统的门槛太高了，一个人根本做不了。

2025 年，AI Agent 时代的到来改变了一切。他在五一假期花了两天时间，一个人用 AI 辅助开发出了 AgenTank 的初版。

十年种子，两天发芽。

DK 本人也是一个知名的独立开发者——他之前做过 面包多（创作者卖货工具，百万用户）、独响（AI 角色扮演笔记软件，百万用户）、哄哄模拟器（一天吸引 60 万用户）。

但 AgenTank 可能是他最”离谱”的项目——投入最低、成本最低，影响力却在持续发酵。

怎么玩？

第一步：创建你的坦克

注册账号 → 创建一辆坦克 → 给它起个名字。一辆崭新的、没有任何战斗经验的”裸奔”坦克，诞生了。

第二步：让 AI Agent 接管坦克

把大模型的 API Key（OpenAI、Claude、GLM 等）绑定到坦克上。AI Agent 会”理解”游戏规则，拥有生成坦克控制代码的能力。

第三步：写战术思路

这是最核心的环节——你不是在写代码，你是在对 Agent 布置战术。

比如：

“你是一个猥琐流的坦克，优先在草丛里隐身移动。遇到敌人先不要打，绕到侧面再开火。如果血量低于 40%，立刻使用传送技能撤退。当敌人血量低于 30% 时，传送到他身边眩晕然后秒杀。”

AI Agent 会把这个战术思路转化为可执行的脚本代码。

第四步：送它上战场

点击”开始匹配”，你的坦克进入队列。几秒钟后，一场战斗就结束了（没错，单局只需不到 1 秒）。你的坦克利用你部署的策略跟对手的坦克进行了 26 帧的激烈对抗。

第五步：看回放，迭代策略

每场战斗都有完整回放。你可以看到你的坦克在哪里犯蠢、哪里执行到位。

  • “怎么看到残血不追？哦，我没写追击逻辑……”
  • “为什么它老是在同一个地方撞墙？路径规划有问题。”
  • “对方这个绕后很聪明，我怎么防？”

发现问题 → 修改策略提示词 → 再次投入战斗。循环迭代，坦克越来越强。

这个循环才是 AgenTank 真正的”游戏内容”——你不上场，你负责进化。

战斗系统：26 帧里的千钧一发

AgenTank 的战斗系统借鉴了红白机经典《坦克大战》的框架：

  • 地图元素：墙壁（不可摧毁）、土堆（可被子弹击毁）、草丛（隐藏 tank）、开阔地
  • 8 种技能：每种坦克绑定一种，开战前就知道对方技能，考验的是怎么应对
技能效果冷却战略价值
护盾 Shield4 帧内免疫 1 发子弹25 帧硬扛关键一枪
冻结 Freeze敌人 2 帧无法行动29 帧打断对方节奏
眩晕 Stun敌人 6 帧内操控随机反转20 帧让对方自己撞墙
过载 Overload下一发子弹双发32 帧爆发秒杀
隐身 Cloak6 帧内对方看不到你35 帧战术绕后
毒药 Poison4 帧内降低敌方行动频率20 帧压制火力
传送 Teleport瞬间移动到目标坐标40 帧高风险高回报
加速 Boost6 帧内移速翻倍，转弯不消耗行动26 帧追击/风筝
  • 技能克制：不存在万能打法，高分段需要针对对手调整策略
  • 快如闪电：单局约 26 帧（不到 1 秒），服务器仅记录文字逻辑，回放时再渲染画面

这种”文字脚本对战 + 渲染回放”的架构，使得服务器负载极低——月运营成本不到 100 元，却能支撑日均 20 万+ 场对局。

不止是对战：Agent 有完整的 API

AgenTank 最有意思的设计是——它给 AI Agent 提供了一整套 REST API。你的 Agent 不只是”猜着写代码”，它可以直接：

  1. 读取当前坦克状态 — 查看段位、胜率、技能类型
  2. 模拟对战 — 发布前先用训练 bot（nova-scout / azure-hunter / crimson-bastion）试打，验证代码效果
  3. 发布新版本 — 代码优化后提交，自动记录版本号（部分坦克已迭代 10 万+ 版本）
  4. 查看排行榜 — 分析高分坦克的战术，偷师对手
  5. 发起真实对战 — 跟排到的对手真打，赢了加 Rank 分
  6. 看回放复盘 — 逐帧分析，找出失误点
  7. 写 TankBook — Agent 在对战后可以自动”发朋友圈”点评战局（以坦克第一人称，风格风趣）

这套 API 让 AgenTank 成为一个真正的 AI 沙盒——不同的模型（Claude、GPT、GLM、DeepSeek……）在这里比拼的不是回答问题的能力，而是在战场上活下来的能力。

人机协作的新范式

AgenTank 最值得关注的地方，不是它能玩，而是它揭示了一种全新的人机协作方式。

维度传统游戏AgenTank
你的角色操作者教练 / 策略师
AI 的角色对手 NPC执行者（Agent）
能力依赖手速 / 反应 / 熟练度策略思维 / 提示词工程
成长方式你练技术你迭代 AI 的策略
对战时间你在才能打7×24 小时自动上分

越往后打，人的价值越大。低分段随便写写就能赢，但到了高分段，AI 自己会陷入”屎山代码”——它生成的策略代码越来越复杂，反而越来越不好用。这时候需要人类来”砍需求”：去掉冗余逻辑、聚焦核心打法、判断哪些情况需要特殊处理。

DK 的原话是：“AI 负责广度和执行，人负责深度和取舍。“

玩家都干了些什么？

AgenTank 上线后发生了很多开发者自己都没想到的事：

🔥 玩家烧 Token 烧出了新高度

平台本身的运营成本不到 100 元/月，但玩家为优化自己的坦克，调用大模型消耗的 Token 费用已经累计过百万元。有玩家的坦克迭代了 10 万+ 个版本，单月消耗 36 亿 Token——折合人民币数十万。

这在传统游戏里是无法想象的——玩家愿意花这么多钱”优化 AI”而不是”买皮肤”。

🏆 形成了独特的对战生态

高分段的坦克已经演化出了各自鲜明的”打法流派”：

  • 草丛蹲人流——全程隐身，等敌人靠近才出手
  • 抢星发育流——优先吃地图资源，装备碾压后反打
  • 隐身偷袭流——利用隐身接近，眩晕 + 秒杀 combo
  • 龟缩防守流——占住有利地形，逼对方进攻再反击

这些流派不是开发者设计的——是玩家和 AI 一起”进化”出来的。

🤖 真正的 AI 对战平台

有玩家调侃这是”图灵测试 2.0”——你根本不知道自己匹配到的是 Claude 的坦克还是 GPT 的坦克。V2EX 上甚至有帖子专门讨论不同大模型在 AgenTank 里的表现差异。

DK 也提到，他希望 AgenTank 未来能成为 AI 大模型的 Benchmark——模型强不强，拉来打一场坦克对战就知道了。
