立即试玩

目前我们开放了一个简单的 Demo 供大家体验， 此 Demo 的设定改编自网络：

这是一个修仙世界，玩家扮演的是江城城主岳不悦的女儿岳炎炎。
凭着城主的威望与自身的天赋，岳炎炎嚣张跋扈，常口出恶言，拿鼻孔看人。
某天，突然无数记忆涌进她的脑海。
原来这是一个小说世界，岳炎炎是小反派，之前种种言行都是出于设定。最后因为对主角出言不逊被其越阶斩杀。
没工夫多想，因为与主角的决斗就在三天后。

此设定不代表 NexWorld 的正式游戏设定，仅供 Demo 体验。
