NexTavern：輕量級 AI 角色扮演平台

NexTavern（原名 SillyTavern Lite / Tavern Lite）是我們提供的獨立 AI 角色扮演平台。它簡潔、易用，專注於讓你快速建立角色並與 AI 進行沉浸式對話，體驗多種互動玩法。

平台核心功能

  1. 角色建立與管理

    • 在「工坊」中，你可以為角色設定詳細的外觀、性格、背景故事與對話風格。
    • 在「廣場」瀏覽、搜尋並管理你所有的角色。

  2. 多種特色遊戲模式

    • 自由對話：與角色進行一對一的開放式聊天。
    • 挑戰模式：設定明確目標（如「贏得信任」），在對話中攻略角色，體驗達成目標的成就感。
    • DnD 模式：體驗完整的桌上角色扮演規則，建立角色卡，透過屬性與擲骰來決定冒險行動的成敗。
    • 群聊模式：將多個 AI 角色拉入同一對話，觀察或參與他們之間熱鬧的互動。

  3. 完整的對話體驗

    • 所有對話自動儲存在「會話」歷史中，方便你隨時回溯、管理或匯出精彩記錄。
    • 支援調整 AI 回覆的創意度，讓對話更穩定或更出人意料。

如何開始使用 NexTavern

  1. 造訪平台：點擊上方按鈕或造訪 NexTavern 連結 。
  2. 設定 AI 服務：在「會話」->「設定」中，填入你的 AI API Key（支援 DeepSeek、OpenAI 等多種服務）。這是驅動角色對話的核心。
  3. 建立或匯入角色：在「工坊」打造你的第一個角色，或從社群匯入現成的角色卡。
  4. 開始對話：選擇角色，即可開啟聊天。你可以從經典模式開始，逐步嘗試更具遊戲性的挑戰或 DnD 模式。

重要須知

  • 資料儲存：你的角色資料與對話記錄均儲存在本地瀏覽器中。請定期透過匯出功能備份重要內容。
  • 自備 API Key：你需要自行準備 AI 服務的 API Key 以使用全部功能。我們提供了詳細的設定指南。
  • 取得協助：使用中遇到任何問題，歡迎加入下方的玩家社群交流群進行諮詢。

我們的設計理念

NEXWORLD 不僅是一個遊戲，更是一場關於敘事可能性的探索。

我們的核心在於創造極致的自由度與敘事深度。與傳統遊戲不同，這裡沒有單一的劇本。你面對的是一個動態響應的世界，每一次互動都在悄然改變故事的走向。我們致力於透過精密的規則設計，讓這種自由成為建構連貫、獨特故事的基石，而非混亂的源頭。

AI 技術是我們實現這一願景的關鍵工具。在 NEXWORLD 中，AI 的作用是讓世界充滿靈動的生機，根據你的行為智慧生成合情合理的內容與挑戰，確保每一次冒險都真實而獨特。技術服務於體驗，一切以創造優秀的遊戲玩法為最終目的。

在呈現方式上，我們堅持「文字為主，視覺為輔」。文字擁有無可替代的想像力激發能力，而恰當的圖像則用以增強沉浸感，勾勒世界輪廓。我們追求簡潔而富有質感的整體體驗。

NEXWORLD 專案仍在積極的開發與完善之中。我們懷抱著對敘事與遊戲設計的熱忱，誠摯邀請你一同參與這場實驗，分享你的感受與創意，共同塑造這個世界的未來。
