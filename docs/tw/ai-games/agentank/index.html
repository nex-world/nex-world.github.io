AI 遊戲推薦AgenTank：AI 坦克大戰

AgenTank：不是你在打遊戲，是你的坦克 Agent 在打

你給 AI 一個”戰術思路”，AI 把它寫成坦克的控制程式碼，然後你的坦克就上戰場了——7×24 小時不眠不休，在你睡覺的時候也在偷偷上分。
450 萬場對局、80+ 王者段位坦克、月營運成本不到 100 元——這是目前最離譜的 AI 原生遊戲之一。

👉 官網（網頁即玩） 

這不是一個”遊戲”

在聊 AgenTank 之前，先把話說清楚：這不是一個傳統意義上的遊戲。

傳統遊戲裡，你按 W 前進、按空格射擊、用滑鼠瞄準——你的操作直接映射為遊戲行為。

AgenTank 不是這樣的。

你的操作是——打字。

  • “打不過就繞後，利用草叢隱身，等敵人技能 CD 好了再打。”
  • “前期苟著發育，看到殘血再衝出去收割。”
  • “如果對方坦克血量低於 30%，直接傳送到他臉上放眩暈。”

你把這些思路寫下來，AI（Claude Opus、GLM 等大模型）把你的話翻譯成坦克的控制腳本——然後你的坦克就上戰場了。

沒錯，你自己不需要上場。你的敵人也不是人——是別人的 AI 坦克。

這就是為什麼玩家管它叫 “賽博鬥蟋蟀”。

🎮

一句話定義：你當教練，AI 當選手，瀏覽器就是賽場。

為什麼它會存在？

這個故事要從開發者 DK 說起。

2016 年，他參加了一個程式碼對戰比賽，用 600 行 Ruby 程式碼寫了一個坦克 AI，打進了十六強。那種”自己的程式碼在戰場上廝殺”的感覺，讓他念念不忘。

2021 年他想復刻這個體驗，但發現寫一個完整的遊戲 + AI 系統的門檻太高了，一個人根本做不了。

2025 年，AI Agent 時代的到來改變了一切。他在五一假期花了兩天時間，一個人用 AI 輔助開發出了 AgenTank 的初版。

十年種子，兩天發芽。

DK 本人也是一個知名的獨立開發者——他之前做過 麵包多（創作者賣貨工具，百萬用戶）、獨響（AI 角色扮演筆記軟體，百萬用戶）、哄哄模擬器（一天吸引 60 萬用戶）。

但 AgenTank 可能是他最”離譜”的項目——投入最低、成本最低，影響力卻在持續發酵。

怎麼玩？

第一步：創建你的坦克

註冊帳號 → 創建一輛坦克 → 給它起個名字。一輛嶄新的、沒有任何戰鬥經驗的”裸奔”坦克，誕生了。

第二步：讓 AI Agent 接管坦克

把大模型的 API Key（OpenAI、Claude、GLM 等）綁定到坦克上。AI Agent 會”理解”遊戲規則，擁有生成坦克控制程式碼的能力。

第三步：寫戰術思路

這是最核心的環節——你不是在寫程式碼，你是在對 Agent 佈置戰術。

比如：

“你是一個猥瑣流的坦克，優先從草叢裡隱身移動。遇到敵人先不要打，繞到側面再開火。如果血量低於 40%，立刻使用傳送技能撤退。當敵人血量低於 30% 時，傳送到他身邊眩暈然後秒殺。”

AI Agent 會把這個戰術思路轉化為可執行的腳本程式碼。

第四步：送它上戰場

點擊”開始匹配”，你的坦克進入佇列。幾秒鐘後，一場戰鬥就結束了（沒錯，單局只需不到 1 秒）。你的坦克利用你部署的策略跟對手的坦克進行了 26 幀的激烈對抗。

第五步：看回放，迭代策略

每場戰鬥都有完整回放。你可以看到你的坦克在哪裡犯蠢、哪裡執行到位。

  • “怎麼看到殘血不追？哦，我沒寫追擊邏輯……”
  • “為什麼它老是在同一個地方撞牆？路徑規劃有問題。”
  • “對方這個繞後很聰明，我怎麼防？”

發現問題 → 修改策略提示詞 → 再次投入戰鬥。循環迭代，坦克越來越強。

這個循環才是 AgenTank 真正的”遊戲內容”——你不上市場，你負責進化。

戰鬥系統：26 幀裡的千鈞一髮

AgenTank 的戰鬥系統借鑒了紅白機經典《坦克大戰》的框架：

  • 地圖元素：牆壁（不可摧毀）、土堆（可被子彈擊毀）、草叢（隱藏 tank）、開闊地
  • 8 種技能：每種坦克綁定一種，開戰前就知道對方技能，考驗的是怎麼應對
技能效果冷卻戰略價值
護盾 Shield4 幀內免疫 1 發子彈25 幀硬扛關鍵一槍
凍結 Freeze敵人 2 幀無法行動29 幀打斷對方節奏
眩暈 Stun敵人 6 幀內操控隨機反轉20 幀讓對方自己撞牆

  • 技能克制：不存在萬能打法，高分區段需要針對對手調整策略
  • 快如閃電：單局約 26 幀（不到 1 秒），伺服器僅記錄文字邏輯，回放時再渲染畫面

這種”文字腳本對戰 + 渲染回放”的架構，使得伺服器負載極低——月營運成本不到 100 元，卻能支撐日均 20 萬+ 場對局。

不只是對戰：Agent 有完整的 API

AgenTank 最有意思的設計是——它給 AI Agent 提供了一整套 REST API。你的 Agent 不只是”猜著寫程式碼”，它可以直接：

  1. 讀取當前坦克狀態 — 查看段位、勝率、技能類型
  2. 模擬對戰 — 發布前先用訓練 bot（nova-scout / azure-hunter / crimson-bastion）試打，驗證程式碼效果
  3. 發佈新版本 — 程式碼優化後提交，自動記錄版本號（部分坦克已迭代 10 萬+ 版本）
  4. 查看排行榜 — 分析高分坦克的戰術，偷師對手
  5. 發起真實對戰 — 跟排到的對手真打，贏了加 Rank 分
  6. 看回放復盤 — 逐幀分析，找出失誤點
  7. 寫 TankBook — Agent 在對戰後可以自動”發朋友圈”點評戰局（以坦克第一人稱，風格風趣）

這套 API 讓 AgenTank 成為一個真正的 AI 沙盒——不同的模型（Claude、GPT、GLM、DeepSeek……）在這裡比拼的不是回答問題的能力，而是在戰場上活下來的能力。

人機協作的新典範

AgenTank 最值得關注的地方，不是它能玩，而是它揭示了一種全新的人機協作方式。

維度傳統遊戲AgenTank
你的角色操作者教練 / 策略師
AI 的角色對手 NPC執行者（Agent）
能力依賴手速 / 反應 / 熟練度策略思維 / 提示詞工程
成長方式你練技術你迭代 AI 的策略
對戰時間你在才能打7×24 小時自動上分

越往後打，人的價值越大。低分段隨便寫寫就能贏，但到了高分段，AI 自己會陷入”屎山程式碼”——它生成的策略程式碼越來越複雜，反而越來越不好用。這時候需要人類來”砍需求”：去掉冗餘邏輯、聚焦核心打法、判斷哪些情況需要特殊處理。

DK 的原話是：“AI 負責廣度和執行，人負責深度和取捨。“

玩家都幹了些什麼？

AgenTank 上線後發生了許多開發者自己都沒想到的事：

🔥 玩家燒 Token 燒出了新高度

平台本身的營運成本不到 100 元/月，但玩家為優化自己的坦克，調用大模型消耗的 Token 費用已經累計過百萬元。有玩家的坦克迭代了 10 萬+ 個版本，單月消耗 36 億 Token——折合人民幣數十萬。

這在傳統遊戲裡是無法想像的——玩家願意花這麼多錢”優化 AI”而不是”買造型”。

🏆 形成了獨特的對戰生態

高分段的坦克已經演化出了各自鮮明的”打法流派”：

  • 草叢蹲人流——全程隱身，等敵人靠近才出手
  • 搶星發育流——優先吃地圖資源，裝備輾壓後反打
  • 隱身偷襲流——利用隱身接近，眩暈 + 秒殺 combo
  • 龜縮防守流——占住有利地形，逼對方進攻再反擊

這些流派不是開發者設計的——是玩家和 AI 一起”進化”出來的。

🤖 真正的 AI 對戰平台

有玩家調侃這是”圖靈測試 2.0”——你根本不知道自己匹配到的是 Claude 的坦克還是 GPT 的坦克。V2EX 上甚至有帖子專門討論不同大模型在 AgenTank 裡的表現差異。

DK 也提到，他希望 AgenTank 未來能成為 AI 大模型的 Benchmark——模型強不強，拉來打一場坦克對戰就知道了。

值得關注的原因

AgenTank 和之前我們在全景報告裡提到的所有遊戲都不同。

對比項崇禎 / AI RogueliteDeath by AIAgenTank
AI 的角色遊戲引擎裁判你的代理（Agent）
你的角色玩家參與者教練 / 策略師
人機關係AI 服務你AI 評判你你指導 AI
核心玩法與 AI 互動與朋友互動訓練 AI 對戰

它是目前唯一一個把 “人機協作”而非”人機對抗” 作為核心玩法的 AI 原生遊戲。
