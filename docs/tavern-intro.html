Tavern 客户端

Tavern 完整用户指南

平台简介

Tavern（SillyTavern Lite）是一个基于AI的角色扮演与聊天平台。在这里，你可以创建各类角色，并与他们进行智能对话。平台的核心是使用大型语言模型（LLM）来驱动角色，使它们能够根据你的设定进行自然、生动的互动。

与简单的聊天机器人不同，本平台提供了多种结构化的“游戏模式”，让你不仅能自由对话，还能体验带有目标、规则和多人互动的沉浸式玩法。无论你是想进行轻松的角色扮演，还是想体验一场完整的奇幻冒险，这里都能满足你的需求。

快速开始

第一步：配置AI模型

  1. 点击顶部导航栏的 “Go” > “Config”。
  2. 在左侧菜单中，进入 “配置” > “LLM 配置”。
  3. 你需要一个API密钥来连接AI服务。我们推荐使用 DeepSeek：访问其官网注册账号，在控制台创建并复制API密钥。
  4. 将密钥粘贴回本平台的“API Key”栏，点击 “Fetch Models” ，然后选择一个模型（如deepseek-chat）。成功后即可进行下一步。

第二步：创建第一个角色

  1. 点击 “新建角色”。
  2. 填写角色基本信息：姓名、详细的外观年龄描述（如：“一位身着黑色风衣的私家侦探，眼神锐利，习惯性皱眉头”）。
  3. 设定性格与背景：定义其人格、说话风格及故事场景。
  4. 为角色写下第一句 开场白。
  5. 保存后，在 “广场” 找到该角色，点击 “开始对话” 即可聊天。

核心功能详解

1. 角色管理

  • 工坊：角色创作与编辑中心。你可以在此精细打磨角色的所有设定。
  • 广场：浏览已创建的所有角色。支持搜索、按标签筛选，并可直接从这里发起对话。

2. 会话管理

所有对话记录都保存在 “会话” 页面。你可以在此继续历史对话、重命名或删除会话，也可以导出精彩的对话记录进行保存。

3. 特色游戏模式

本平台除了基础的自由聊天外，还提供了三种独特的玩法，这是我们的核心特色。

模式一：挑战模式 - 为目标而对话

这是一种带有明确攻略目标的角色扮演。你需要在开始前设定一个挑战，例如“说服国王同意出兵”或“在宴会上查明真凶”。

  • 如何玩：系统会根据你的挑战目标，在后台设定一些隐藏的状态变量（如“信任值”、“嫌疑度”）。你与角色的每一轮对话，都会由系统智能评估，并影响这些变量。
  • 玩点：你会在对话中收到系统关于进展的微妙提示（例如：“国王看起来有所动摇，但尚未下定决心”）。通过策略性地选择对话内容，引导变量向目标迈进，最终达成成功（或失败）的特定结局，带来强烈的成就感和故事驱动感。
模式二：DnD模式 - 桌面角色扮演体验

这是对《龙与地下城》等桌面角色扮演游戏的完整数字模拟，提供规则化的沉浸冒险。

  • 如何玩：首先，你需要创建一张包含“力量、敏捷、智力”等六维属性的角色卡。游戏由AI扮演的“地下城主”（DM）引导叙事。
  • 核心机制 - 骰子检定：当你决定角色要行动时（如“潜行绕过守卫”或“说服商人”），系统会投掷虚拟的20面骰（D20），并结合你的角色属性来决定成败。DM会根据结果描述出人意料的剧情发展。
  • 玩点：体验真正的回合制探索、解谜和战斗。骰子带来的随机性让每次行动都充满惊喜，复杂的角色成长和策略配合提供了深度和重玩价值。
模式三：群聊模式 - 多人角色茶话会

在此模式中，你可以将多个AI角色聚集到同一个对话中，观察或参与他们之间的互动。

  • 如何玩：从角色库中选择多个角色创建群聊。你可以随时以“自己”的身份发言，也可以静观其变。
  • 玩点：系统会智能地安排角色之间的发言顺序，让对话自然、流畅地进行。你能看到不同性格的角色碰撞出戏剧性的火花，自发衍生出有趣的剧情，非常适合轻松社交和创意激发。

优化你的体验

  1. 调整AI回复风格：在LLM 配置中，尝试调整 “温度 (Temperature)” 参数。较低的数值（如0.2）使回复更稳定；较高的数值（如0.8）使回复更富创意和变化。
  2. 优化角色表现：如果AI的回复不符合预期，请回到“工坊”编辑该角色，补充更详细的性格描述和具体的对话示例，这能显著提升AI的扮演精度。
  3. 重要数据备份：请定期通过“工坊”的导出功能备份你珍视的角色卡，并通过“会话”列表的导出功能备份精彩的对话记录。

常见问题与解决

  • AI无响应或报错：首先检查LLM 配置中的API密钥是否正确且未过期，并确保网络连接正常。
  • 回复内容偏离角色：尝试为角色添加更明确的“系统提示词”，例如“请始终以冷酷的语气说话”。同时，检查并完善角色的“人格设定”部分。
  • 如何重试某条回复：在对话界面，将鼠标悬停在AI的回复上，通常会显示“重新生成”按钮，点击即可获得一个新的回复版本。
  • 页面运行缓慢：尝试清理浏览器缓存，或关闭一些暂时不用的对话标签页。

最后建议

最好的学习方式是立即实践。从创建一个简单的角色开始，体验基础聊天的乐趣。然后，逐步尝试挑战模式，感受达成目标的快感；或者组建一个群聊，享受热闹的互动。当你准备好时，踏入DnD模式，开始一场真正的传奇冒险。

祝你在 Tavern 的世界里创造无数精彩的故事。
