API Key 配置指南

API Key 配置完全指南

什么是 API Key？

API Key 就像是你的”AI 服务通行证”。NEXWORLD 和 NexTavern 需要通过它来调用 AI 服务，让游戏角色能够智能对话。

简单理解：API Key = 你和 AI 服务之间的密码

🆓 推荐的免费/低价 API 提供商

方案 1：OpenAI 官方（推荐新手）

  • 优点：稳定可靠，首次注册送 $5 免费额度
  • 缺点：需要国际信用卡
  • 获取地址：https://platform.openai.com/api-keys 
  • 价格：$0.002/1K tokens（约可使用 2500 次对话）

方案 2：国内 API 中转服务

  • 优点：支持支付宝/微信，无需信用卡
  • 推荐平台：
    • API2D：https://api2d.com  （新用户送 10 元）
    • CloseAI：https://console.closeai-asia.com 
    • OpenAI-SB：https://openai-sb.com 

方案 3：免费试用平台

  • Anthropic Claude：https://console.anthropic.com  （部分地区免费）
  • Google Gemini：https://ai.google.dev  （有免费配额）
  • DeepSeek：https://platform.deepseek.com  （国产，价格低）

📋 获取 API Key 步骤（以 OpenAI 为例）

第 1 步：注册账号

  1. 访问 https://platform.openai.com 
  2. 点击 “Sign up” 注册账号
  3. 验证邮箱

第 2 步：获取 API Key

  1. 登录后，点击左侧 “API keys”
  2. 点击 ”+ Create new secret key”
  3. 输入名称（如 “NEXWORLD”）
  4. 重要：立即复制 API Key（只显示一次！）

格式示例：sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx

第 3 步：充值（可选）

  • 首次注册有 $5 免费额度
  • 用完后可充值 $10-$20（够用很久）

文字步骤

  1. 访问 NEXWORLD Demo 
  2. 向下滚动到页面底部
  3. 找到并点击 「模型配置」 折叠面板
  4. 在 “API Key” 输入框中粘贴你的密钥
  5. 选择对应的 API 提供商（如 OpenAI）
  6. 点击”保存”

配置成功标志：可以开始对话，AI 能正常回复

💬 在 NexTavern 客户端中配置

  1. 访问 NexTavern 客户端 
  2. 点击右上角 ⚙️ 设置
  3. 选择 “API Settings”
  4. 填写以下信息：
    • API Type: 选择你的服务商（如 OpenAI）
    • API Key: 粘贴你的密钥
    • API URL: （通常自动填写）
  5. 点击 “Test Connection” 测试连接
  6. 显示 ✅ 成功后保存

❓ 常见问题 FAQ

Q1: API Key 会被泄露吗？

A: 不会。你的 API Key 只保存在你的浏览器本地，不会上传到 NEXWORLD 服务器。

Q2: API Key 用完了怎么办？

A:

  • 检查余额：访问 https://platform.openai.com/usage 
  • 充值：在账户页面添加付款方式
  • 或者：换一个免费试用平台

Q3: 配置后仍无法使用？

检查清单：

  • ✅ API Key 格式正确（sk-开头）
  • ✅ 账户有余额
  • ✅ 选择了正确的 API 提供商
  • ✅ 网络连接正常

Q4: 使用成本大概多少？

参考消费：

  • 轻度使用（每天 10 次对话）：约 $1-2/月
  • 中度使用（每天 50 次对话）：约 $5-10/月
  • 重度使用（每天 200+ 次）：约 $20-30/月

Q5: 可以多人共用一个 API Key 吗？

A: 技术上可以，但不推荐（会很快用完额度且不安全）

🔒 安全提示

  • ⚠️ 不要分享 API Key 给任何人
  • ⚠️ 不要截图包含 API Key 的内容发到网上
  • ⚠️ 定期更换 API Key（在控制台可重新生成）
  • ⚠️ 设置用量限制（在 API 控制台设置月度上限）
