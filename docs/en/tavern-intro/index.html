NexTavern Complete User Guide

Platform Overview

NexTavern (formerly Tavern Lite) is an AI-powered roleplay and chat platform. You can create characters of all kinds and engage in intelligent conversations with them. The platform uses large language models (LLM) to drive characters, enabling natural and vivid interactions based on your settings.

Unlike simple chatbots, this platform offers structured “game modes” that let you do more than just chat freely — experience immersive gameplay with goals, rules, and multi-character interactions. Whether you want light roleplay or a full fantasy adventure, there’s something here for you.

Quick Start

Step 1: Configure the AI Model

  1. Click “Go” > “Config” in the top navigation bar.
  2. In the left menu, go to “Settings” > “LLM Configuration”.
  3. You’ll need an API key to connect to an AI service. We recommend DeepSeek: sign up on their website, create an API key in the console, and copy it.
  4. Paste the key into the “API Key” field, click “Fetch Models”, then select a model (e.g., deepseek-chat). Once successful, proceed to the next step.

Step 2: Create Your First Character

  1. Click “New Character”.
  2. Fill in basic info: name, detailed appearance and age description (e.g., “A private detective in a black trench coat, sharp-eyed, with a habit of furrowing his brow”).
  3. Set personality and background: define their personality, speaking style, and story setting.
  4. Write the character’s first opening line.
  5. Save, then find the character in the Square and click “Start Chat”.

Core Features

1. Character Management

  • Workshop: The character creation and editing hub. Fine-tune every detail of your character’s settings here.
  • Square: Browse all created characters. Supports search, tag filtering, and starting conversations directly.

2. Session Management

All conversation history is saved in the Sessions page. You can resume past conversations, rename or delete sessions, and export memorable dialogues for safekeeping.

3. Featured Game Modes

Beyond basic free chat, the platform offers three unique gameplay modes — our core differentiator.

Mode 1: Challenge Mode — Dialogue with a Goal

This is roleplay with a clear objective. Set a challenge before starting, such as “Convince the king to send troops” or “Uncover the real culprit at the banquet.”

  • How it works: The system creates hidden state variables (e.g., “trust level”, “suspicion level”) based on your goal. Each round of dialogue with the character is evaluated by the AI, affecting these variables.
  • What makes it fun: You receive subtle progress hints (e.g., “The king seems swayed, but hasn’t made up his mind”). Strategically choose your dialogue to steer variables toward your goal, reaching a success (or failure) ending — delivering a strong sense of achievement and story drive.
Mode 2: DnD Mode — Tabletop Roleplay Experience

A full digital simulation of tabletop RPGs like Dungeons & Dragons, offering rule-based immersive adventures.

  • How it works: First, create a character sheet with six attributes (Strength, Dexterity, Intelligence, etc.). The game is narrated by an AI Dungeon Master (DM).
  • Core mechanic — Dice Checks: When your character takes an action (e.g., “sneak past the guards” or “persuade the merchant”), the system rolls a virtual 20-sided die (D20) and combines it with your attributes to determine success or failure. The DM describes the unfolding story based on the result.
  • What makes it fun: Experience genuine turn-based exploration, puzzle-solving, and combat. Dice randomness makes every action a surprise. Deep character progression and strategic party composition offer replayability and depth.
Mode 3: Group Chat Mode — Multi-Character Party

In this mode, you can gather multiple AI characters into a single conversation, observing or participating in their interactions.

  • How it works: Select multiple characters from your library to create a group chat. You can chime in as “yourself” at any time, or just watch the drama unfold.
  • What makes it fun: The system intelligently manages turn order for natural, fluid conversation. Watch different personalities clash, creating dramatic sparks and emergent storylines — perfect for lighthearted social fun and creative inspiration.

Optimizing Your Experience

  1. Adjust AI Response Style: In LLM Configuration, try adjusting the “Temperature” parameter. Lower values (e.g., 0.2) produce more stable responses; higher values (e.g., 0.8) yield more creative and varied outputs.
  2. Improve Character Performance: If the AI’s responses don’t match expectations, go back to the Workshop and edit the character. Add more detailed personality descriptions and specific conversation examples — this significantly improves roleplay accuracy.
  3. Backup Important Data: Regularly use the Workshop export function to back up your treasured character cards, and the Sessions export function to save memorable conversations.

FAQ & Troubleshooting

  • AI not responding or showing errors: First check that your API key in LLM Configuration is correct and hasn’t expired. Ensure your internet connection is working.
  • Responses straying from character: Try adding clearer “system prompts” for the character, e.g., “Always speak in a cold tone.” Also review and refine the character’s “Personality” section.
  • How to retry a response: Hover over the AI’s response in the chat interface — a “Regenerate” button usually appears. Click it to get a new version.
  • Page running slowly: Try clearing your browser cache or closing unused chat tabs.

Final Advice

The best way to learn is to start practicing right away. Create a simple character and enjoy basic chat. Then, gradually try Challenge Mode for the thrill of achieving goals, or set up a Group Chat for lively interactions. When you’re ready, step into DnD Mode and begin a true legendary adventure.

May you create countless wonderful stories in the world of NexTavern.
